# melis-commerce

Melis Platform Commerce module

## Requirements

- PHP 7.0+

## Installing

This project can be installed using Composer. Add the following to your
composer.json:

```javascript
    {
        "require": {
            "melisplatform/melis-commerce": "dev-master"
        }
    }
```
